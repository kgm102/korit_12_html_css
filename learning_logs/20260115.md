- 저희가 Java를 배우면서 작성한 것은 code라는 표현을 썼지만, HTML + CSS

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>document</title>
</head>
<body>
    <h1>Hello, World!</h1>
    예를 들어 여기서 계속 작성하면 옆에 preview 반
    영이 됩니다.
</body>
</html>
```
